# PIN PAD Project

Этот пет-проект представляет собой простое веб-приложение, имитирующее работу с ПИН-код панелью. Этот проект представляет собой реализацию ПИН-пада (клавиатуры для ввода ПИН-кода). Он предназначен для ввода и проверки цифровых комбинаций. 

### Основные возможности:
- Ввод ПИН-кода: обработка нажатий кнопок на матричной клавиатуре 4x4;
- Проверка ПИН-кода: сравнение введенной последовательности с заданным числом;
- Визуальная обратная связь: индикация успешного ввода или ошибки (изменяющийся фон приложения);
- Удаление введенных символов;
- Запрет на ввод больше 4 символов.
